# How-to guides

How-to guides are goal-oriented. They assume you already know the basics and want to accomplish a specific task.

| Guide | Goal |
|---|---|
| [Store snapshots on S3](store-on-s3.md) | Redirect snapshot storage to an S3 bucket |
| [Update snapshots](update-snapshots.md) | Overwrite stored snapshots after an intentional change |
| [Test parametrized cases](parametrized-tests.md) | Track each parameter combination as its own snapshot |
| [Override the snapshot fixture](override-fixture.md) | Change the extension or location for a specific test suite |
